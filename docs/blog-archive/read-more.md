The Read More section provides settings for customizing the appearance of the
Read More link on archive pages. Below is a screenshot of what the Read More
link looks like:

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5a03f50f2c7d3a272c0d866f/images/5cd3e8c72c7d3a177d6e9029
/file-bm5qH1eox9.png)

## Display

Changes the display style of the read more link. There are 3 options:

  1.  **Inline** \- Keeps the link inline with the entry content.
  2.  **Block** \- Breaks the link onto a new line (recommended if "Button" style is selected).
  3.  **None** \- Hides the read more link completely.

## Style

There are 2 styles to choose from, link and button. Below is a screenshot of
what the button style looks like:

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5a03f50f2c7d3a272c0d866f/images/5cd3e8bc04286306738ee8ed

Text

Changes the text of the read more button. Some examples would be "Continue
Reading" or "View Post".

Show Ellipses
